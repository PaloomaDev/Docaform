1-Google Forms configuration

Add questions in the Google Forms. 

Warning : question types available with FormsDocGo are 

  • Short Answer
  • Paragraph
  • Multiple choice
  • Dropdown

More information here Choose a question for your form

2-Select a Google Docs as template

One the Google Forms questions are added, open the Addon menu and click on Google Docs template. 

You will be able to select a Google Docs as a template only for this Google Forms.

When an user will answer the forms, FormsDocGo will copy this template and add the user's answers in the template copy depeding on how configured the template.

3-Google Docs template configuration

To configure the Google Docs template : 

  1. Open the Addon Menu and click on Matching Table . This table will give you the label for each questions or question/choice duet to use in the Google Docs template
  2. Once this table is displayed, you can place your label in the Google Docs template 

 

Example
